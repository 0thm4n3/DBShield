# DBShield

Protects your data by inspecting incoming queries from your application server and rejecting abnormal ones. it also supports data masking to avoid data leaks.

# Supported Databases:

| Database      | Protect       | Data Masking  |
| ------------- |:-------------:| -------------:|
| MariaDB       | Yes           |      Yes      |
| MySQL         | Yes           |      Yes      |
| Oracle        |     Yes       |      No      |
